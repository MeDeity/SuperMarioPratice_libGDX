#SuperMarioPractice

Use [libGDX](https://libgdx.badlogicgames.com/) to create SuperMario game

Fundamental ideas are learned from [Brent Aureli](https://www.youtube.com/watch?v=a8MPxzkwBwo)

##Controls:
LEFT / RIGHT / DOWN to control Mario movements

C to jump

X to run / fire
